# Obsidian Android Automation (ADB + Python)

This project demonstrates Android UI automation for the Obsidian mobile app using
Python and ADB, following a multi-agent execution approach.

## Features
- Launch Obsidian app via ADB
- Automate note creation and text entry
- Verify data persistence across app restarts
- Backup Obsidian vault from emulator storage
- Agent-based execution design
